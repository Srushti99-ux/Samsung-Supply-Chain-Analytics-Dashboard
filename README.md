# Samsung-Supply-Chain-Analytics-Dashboard

End-to-end Supply Chain Analytics Dashboard built using Power BI analyzing procurement, inventory, shipment, and sales performance.

📊 Project Overview

This project presents an end-to-end Supply Chain & Logistics Analytics Dashboard built using Power BI.
The dashboard analyzes procurement, supplier performance, inventory management, shipment efficiency, and sales performance to provide data-driven business insights.

⚠️ Important note: This is NOT an official Samsung dataset — the data is synthetic / practice-based, created purely for learning and skill development.

🎯 Business Objective

To monitor and optimize:
✔ Supplier lead time & quality performance
✔ Inventory stock levels & defect rate
✔ Shipment delays & carrier performance
✔ Revenue, profit margin & discount analysis

🧩 What the dashboard solves

i. Identify inventory risks (overstock, safety stock, reorder points)
ii. Track supplier performance (cost, lead time, quality)
iii. Analyze shipment delays & delivery efficiency
iii. Monitor revenue, profit, margins, and customer demand
iv. Support management in moving from reactive decisions → proactive planning

📌 Key Business Insights:
 • Evaluated supplier efficiency using lead time metrics
 • Assessed inventory health via Turnover Rate & Days of Inventory (DOI)
 • Identified defect trends across products
 • Analyzed shipment delays & logistics bottlenecks
 • Tracked revenue contribution by sales channels
 
📈 Key KPIs Tracked
 • Total Revenue: 176.95M
 • Profit: 48.56M
 • Profit Margin: 27.44%
 • Perfect Order Rate: 75%
 • Total Shipments: 7.5K
 • Total Delay Shipments: 573

🛠 Tools & Skills Applied:
 Power BI | DAX | Data Modeling | KPI Development | Data Visualization

⚠️ Data Disclaimer (Learning Insight)
Since this is a practice dataset, a few KPIs (for example, inventory turnover rate / turnover-related metrics) may not be 100% business-accurate.
This is a known drawback of using synthetic data, but it still provides strong hands-on experience in:
 1. KPI logic building
 2. DAX calculations
 3. Business interpretation
